How a blog admin can ban blog as well as delete a blog comment?

A user with 'Blog Admin' privilege can make a 'Blog post' or 'Comment' banned. Also a user who post a blog or comment on a blog post can delete their respective post or comment. Also a user with Blog Admin privilege can mark the comment or post as spam. Refer the figures below.

Give Blog Admin privilege for user. (Go to manage users, select the user and click on Edit Privilege)

Select the 'Blog Admin' privilege. Now this user will be able to mark the blog post or comment as spam.

Once the user logs in, the user will be able to see the Blog posts if the user clicks on the blog module.

Click on the blog post and the Blog Admin can see the spam link at the top right corner.

Once the blog post is marked as banned, only the user with Blog Admin privilege can un-ban the post. Once the post is banned no user can see the post.

How to Delete a Comment: A user who puts the comment or a user with 'Blog Admin' privilege can delete the comment. Ref Fig 4
