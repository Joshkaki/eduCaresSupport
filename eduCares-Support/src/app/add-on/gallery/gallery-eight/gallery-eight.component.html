How to create a Private Album

Private Album: Private albums will only be visible to the active users who are chosen while creating the album. These albums can be made accessible to particular students and employees.

Steps:

  1. At the top of your dashboard, click the module access icon > Collaboration > Gallery
  2. In the top right, click Create Album button to create an album.
  3. Give the album name in the “Album Name” text field.
  4. Select theAlbum visibility as Private Album.
  5. Select the Active student user as:
    1. Select the batch using the checkbox.
    2. Click on the Select button to see the student list.
    3. Click on the CheckBox to select the student from the list,
    4. Click on Done to confirm the selection.
    Similarly, the user can select the Employee as:
    1. Select the Department by clicking on the Checkbox.
    2. Click on Select button to show the employee list in that particular department,
    3. Select the Employee by clicking on the CheckBox.
    4. Click on the Done button to confirm the selection.
  6. Click on the Create album button to create the album.
