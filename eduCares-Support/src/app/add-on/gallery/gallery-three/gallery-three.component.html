How can I delete or edit an image in an album?

  1. At the top of your dashboard, click the module access icon > Collaboration > Gallery
  2. Place the mouse cursor over the photo in the album, use the edit option.
  3. Click on the delete option to delete the photo and click on Delete Image
  4. Click on Edit Description and provide new description for the selected image.

Multiple images present in an album can be deleted by using

  1. Select More option present at the right corner of the selected album
  2. Select the images present in the album needed to be deleted.
  3. Click the Delete Image to delete the selected images.
