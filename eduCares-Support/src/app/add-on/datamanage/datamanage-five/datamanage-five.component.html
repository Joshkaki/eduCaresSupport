How can I edit or delete input values under category?

At the top of your dashboard, click the module access icon > Data and Reports > Data Management

  1. Click on the category name from the category list.
  2. Perform Edit or Delete operation to update the input values in the field as well as delete fields.
  3. Update the respective fields and click on Save.

The Input values can be edited for each of the field by clicking on edit link against that field. The field can be deleted by clicking on Delete option as shown above.
