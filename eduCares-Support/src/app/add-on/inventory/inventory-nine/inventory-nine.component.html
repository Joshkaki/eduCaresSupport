How to create store?

Similar to store categories, stores can be classified based on the type of items sold or issued. For example, a ‘Stationary’ store, a ‘Sports’ store, or a ‘Canteen’ store.

  1. At the top of your dashboard, click the module access icon > Administration > Inventory > Store Type.
  2. In the Name field, enter the name to identify the item type.
  3. In the Code field, enter the store type code. The code for each store type must be unique.
  4. Click the Save button.
