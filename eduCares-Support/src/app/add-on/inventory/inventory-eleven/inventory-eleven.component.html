How to create an item category?

Item categories are used to group store items together. Item categories are only used when recording a sellable item.

  1. At the top of your dashboard, click the module access icon > Administration > Inventory > Item Category.
  2. In the Name field, enter the name of the item category.
  3. In the Code field, enter the item category code. The code for each item category must be unique.
  4. Click the Save button.
