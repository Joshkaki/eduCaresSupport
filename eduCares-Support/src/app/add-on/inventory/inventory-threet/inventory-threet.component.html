How to add items to store?

A store can contain any number of items. The items may be meant for sale or meant to be indented (issued).

  1. At the top of your dashboard, click the module access icon > Administration > Inventory > Store Items.
  2. In the top right, click the New button.
  3. Select a store.
  4. In the Item Name field, enter the name of the item.
  5. In the Code field, enter an item code.
  6. In the Quantity field, enter the quantity available.
  7. In the Unit Price field, enter the price of one unit of the item.
  8. In the Tax (%) field, enter the percentage of tax applied when purchasing the item.
  9. In the Batch No. field, enter the item batch number.
  10. Select the Sellable checkbox if this item can be sold.
  11. In the Item Category field, select the item category.
    Note: The field is available only if you select the Sellable checkbox.
  12. Click the Save button.
