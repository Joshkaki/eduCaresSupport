How to edit or delete an item category?

An item category can be edited at any time. An item category can be deleted only if it has not been used when adding an item to a store.

  1. At the top of your dashboard, click the module access icon > Administration > Inventory > Item Category.
  2. Click Edit to edit an item category or click Delete to delete an item category.
