How to edit or delete item details?

Item details can be edited at any time. An item can be deleted from a store only if the item has not been indented (issued).

  1. At the top of your dashboard, click the module access icon > Administration > Inventory > Store Items.
  2. Select the name of the store, or you can enter the item name to search. You can also use the ‘Select all’ option to view all items in all stores.
  3. Click Edit to edit item details or click Delete to delete an item.
  4. Click the PDF Report button to generate a PDF report of the list of items or click the CSV Export button to generate a CSV report of the list of items.
