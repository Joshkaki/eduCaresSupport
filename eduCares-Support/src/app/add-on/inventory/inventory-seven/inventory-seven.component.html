How to create store category?

Stores can be categorized based on any attribute. For example, you can categorize stores based on who the items are being sold or issued to, like a ‘Student’ store category, an ‘Employee’ store category, or a ‘Guest’ category. You can also categorize stores as ‘Consumable Items’ and ‘Non-consumable Items’. Or you can categorize stores as a ‘Sellable Store’ category (stores with items for sale) and an ‘Indent Store’ category (stores with items that can be indented).

  1. At the top of your dashboard, click the module access icon > Administration > Inventory > Store Category.
  2. In the Name field, enter the name of the store category.
  3. In the Code field, enter the store category code. The code for each store category must be unique.
  4. Click the Save button.
