How to edit or delete supplier type?

A supplier type can be edited at any time. A supplier type can be deleted only if it has not been used when adding suppliers.

  1. At the top of your dashboard, click the module access icon > Administration > Inventory > Supplier Type.
  2. Click Edit to edit a supplier type or click Delete to delete a supplier type.
