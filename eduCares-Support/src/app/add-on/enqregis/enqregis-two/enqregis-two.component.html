How can I set up course for online registration?

A course must be set up before it is made available for online registration. Click the module access icon Menu > Academics > Applicant Registration > Manage Courses button > Add Course button to open the Add Course page. A number of options are available to help you define the admission requirements.

Field Description
Course Select a course. Only courses created on the Manage Course page (module access icon > Administration > Settings > Manage Course/Batch > Manage Course) are listed.
Minimum score Enter the minimum score required for admission into this course.
Subject based registration Active – When registering for a course, applicants can choose the elective subjects they want to learn.
Inactive – When registering for a course, applicants cannot choose the elective subjects. Applicants must take all the elective subjects of the course they are registered for by default.
Max Electives & Min Electives Enter the maximum and minimum number of elective subjects the applicant can select. These fields are available only if Subject based registration is Active.
Subject based collection Active – When registering for a course, applicants must pay the amount assigned for each subject. Inactive – Applicants must pay the one-time amount that is specified in the Amount field. This field is available only if Subject based registration is Active.
Amount Enter the one-time amount the applicant needs to pay as application fees. This field is available only if Subject based collection is Inactive.
Is active Active – This course will be immediately available for online registration after set up. Inactive – This course will not be available for online registration after set up. Inactive courses can be made active at a later time.
Is pin system available Active – Allow students to register to courses using PIN numbers. The PIN system involves generating a set of PIN numbers that are issued to the applicants to register for a course. Registration cannot be done without a valid PIN number. Inactive – If a course does not use the PIN system, any number of applicants can register for the course.
Enable Approval System Active – When allotting an applicant to a course, the employee handling admissions must verify and mark the student as “academically cleared” and “has paid all fees”. Without clearing these two requirements, the applicant cannot be allotted to a course. Inactive – The applicant can directly be allotted to a course without the need to clear any admission requirements.
Include Additional Details Active – Ability to select the additional admission fields that is to appear in the online admission form. Inactive – Do not include the additional admission fields in the online admission form.
Transfer attachments while allotting Select this checkbox if you want the documents (attachments) submitted by the students to be added to their student profile when allotted to a course.

After you have defined the admission requirements, click the Create button.
