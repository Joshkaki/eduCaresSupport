How can I return a book in Library?

  1. At the top of your dashboard, click the module access icon > Academics > Library > Return book to open the Return book page.
  2. In the Book Number/Barcode field, enter the book number or scan the book’s barcode and then click the Search button.
  3. When the user selects the book’s title, the details of the borrowed book are displayed.  
  4. If the book’s due date has passed, enter the fine amount that is to be collected. This is not a mandatory field. Fines only apply to students, not to employees. All fines paid can be viewed and reverted from the Library fines page.
  5. Click the Return button.
