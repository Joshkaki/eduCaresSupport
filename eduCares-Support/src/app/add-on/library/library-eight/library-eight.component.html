How can I check for the availability of a book?

To quickly check the availability of a book, use the Book search page. You can search for a book using the:

  1. Book’s barcode
  2. Book number
  3. Book’s title
  4. Book’s tag
  5. Book’s author


At the top of your dashboard, click the module access icon > Academics > Library > Search book to open the Book search page.

  1. In the Search field, enter the keyword based on the selected search mode.
  2. Click the Search button.

If you want to issue or reserve the book, click on the book title.

  • Click the Reserve this book button to reserve the book or click the Issue this book button to issue the book to a student or employee.
