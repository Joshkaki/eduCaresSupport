How can I revert/ view library fine in a library?

All fines paid by students can be viewed and reverted from the Library fines page.

  1. At the top of your dashboard, click the module access icon > Academics > Library > Library Fine Receipts to open the Library fines Receipts page.
  2. In the Payer field, enter the name of the student and then specify a date range. This will display corresponding library fine transactions details.
  3. Click the ‘Revert’ icon to revert the fine. The Reverted library fines are displayed in the Reverted transactions page in the Finance module (module access icon > Administration > Finance > Transactions > Reverted transactions). The Administrator and a Privileged Employee (Revert Transaction privilege is required) can be view the reverted library fines.
