How can I add custom fields to record vehicle additional details?

At the top of the dashboard, click the module access icon > Administration > Transport > Settings>Vehicle additional details to open the Vehicle additional details page.

Apart from the default fields available while adding vehicle details, you can add custom fields to record any additional details about the vehicle.

Example: Insurance expiry date, Vehicle model etc


  1. In the Name field, enter the name of the field that will be used when entering vehicle information.
  2. Specify the status of the additional field:
    • Active – An additional detail field with this status will be available when entering vehicle information.
    • Inactive – An additional detail field with this status will not be available when entering vehicle information.
      Note: You can inactivate an additional detail field at any time.
  3. Select the Is mandatory checkbox to set this field as a mandatory field that must be filled when adding a new vehicle.
  4. In the Input method field, select the required field input method from the following options:
    • Text Box – Enter brief text information.
    • Select Box – Select one option from a drop-down list.
    • Check Box – Select one or more items from a set of options.
  5. Click the Create button.
  6. Use the arrow icons to determine the order that the fields will appear when entering vehicle information.
