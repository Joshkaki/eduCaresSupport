How to do the basic settings in Transport?

Complete the basic configuration before start with the Transport setup from,
Home -> Administration -> Transport -> Settings -> Basic Settings


I) Route Mode

Here you can decide how the transport routes are assigned to passengers. There are two option available. You can select any of it according to the institutional requirement.

1- Passengers may be assigned a different pick up and drop route

You can select this option, if passengers are using different pick up and drop route. So while assigning transport to Student/Employee you will get the options as shown below,


2- Passengers are assigned one route for pick and drop

If passengers are using same route for pick up and drop you can enable this option. So while assigning transport, you will get the page like below,


II) Fare Mode

Here you can configure the fare mode of the transport facility. Two options are available here,

1-Stops may have different fares

If each stops have different fares you can select this option. So while creating routes you will get an option to provide different fare for each stops.


2- Route has a single fare for all stops

You can enable this option, If the all stops in a route have a single fair. So, while creating routes you will get an option to update the fare as shown in fig,


III) Auto update Transport fees

You can decide whether to enable or disable auto update of unpaid fee collections when passenger route is changed.

IV) Academic Year

To refer the transport details (Stops, Routes, Vehicles etc) of each academic year, you can select the academic year using this option and check it from the module.
