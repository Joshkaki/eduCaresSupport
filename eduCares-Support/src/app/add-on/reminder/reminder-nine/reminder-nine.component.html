How can I edit or delete reminder settings for an event?

Custom reminders set for events can be edited or deleted as long as the event has not passed. Once the event has passed, the reminders cannot be edited or deleted.

  1. At the top of your dashboard, click the module access icon > Administration > Reminders > View Custom Reminders to open the Custom Reminder page.
    • Custom reminders set for upcoming events are displayed. Use the filter on the right to view the reminders that were set for past events.
  2. Click Edit to edit the reminder settings for an event.
  3. Click Delete to delete the reminder settings for an event.
