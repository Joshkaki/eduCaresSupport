How can I view and edit event reminders?

Use the various options at the top of the View event Reminders page to filter and view the list of events.

  • View the list of events with or without reminders.
  • Use the checkboxes to filter and view specific type of events.
  • View the list of upcoming or past events.
  1. At the top of your dashboard, click the module access icon > Administration > Reminders > View Event Reminders to open the View Event Reminders page.
    TheView Event Reminders page contains preconfigured reminder settings that can be changed.
  2. To edit the settings for an event, click Settings beside the event.
  3. Make the necessary changes, and then click the Save changes button.
