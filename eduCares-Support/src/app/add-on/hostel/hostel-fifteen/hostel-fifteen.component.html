How to assign an existing hostel fee collection to a student?

Previously created fee collections can be assigned to students. At the top of your dashboard, click the module access icon > Administration > Hostel > Fee Collection > User-wise Fee Collections to open the Fee Collections page.

  1. Select the Choose from existing option.
  2. In the Select a batch field, select a batch.
  3. In the Select Fee Collection field, select a fee collection.
  4. In the Search Student field, enter the first name or admission number of the student to search and assign.
    Note: You can only search for students who have been assigned to a hostel.
  5. Click the Create button.
