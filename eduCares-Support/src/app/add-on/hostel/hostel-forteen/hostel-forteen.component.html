How to schedule hostel fee collection for a batch?

  1. At the top of your dashboard, click the module access icon > Administration > Hostel > Fee Collection > Batch-wise Fee Collections.
  2. In the Name field, enter the name of the fee collection.
  3. In the Start date field, select the date from which you intend to start collecting the fees.
  4. In the End date field, select the last date within which the students would be required to pay the fees.
  5. Select the batches.
  6. Click the Create button.
