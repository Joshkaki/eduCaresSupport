How can I add custom fields to record hostel room information?

At the top of your dashboard, click the module access icon > Administration > Hostel > Manage room additional details to open the Manage room additional details page. Here four fields will be available to generate additional details in Hostel, namely Name, Status, Is mandatory and the input method fields. The below information will give you detailed explanation on the same:

  1. In the Name field, enter the name of the field that will be used when recording hostel room details.
  2. Specify the status of the additional field:
    • Active – An additional detail field with this status will be available when recording hostel room details.
    • Inactive – An additional detail field with this status will not be available when recording hostel room details.
    Note: You can inactivate an additional detail field at any time.
  3. Select the Is mandatory check-box to set this field as a mandatory field that must be filled when recording hostel room details.
  4. In the Input method field, select the required field input method from the following options:
    • Text Box – Enter brief text information.
    • Select Box – Select one option from a drop-down list.
    • Check Box – Select one or more items from a set of options.
  5. Click the Create button.
  6. Use the arrow icons to determine the order that the fields will appear when recording hostel room details.
