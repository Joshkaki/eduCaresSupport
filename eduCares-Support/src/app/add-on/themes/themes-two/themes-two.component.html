What is the use of color theme module in eduCares?

With this module the color theme of Fedena can be changed. Go to General Settings under settings module. There is an option to Select Theme. Ref image below . Click on select theme drop down menu to choose the color. It is also provided with More colors option to choose further colors.

Once you select the color it will get reflected in the theme and click on Update.
