How to create a Task?

  1. At the top of your dashboard, click the module access icon > Collaboration > Tasks.
  2. In the top right, click the New Task button.
  3. In the Title as well as description field, enter the name of the task as well as brief description. These are mandatory fields.
  4. Attach file if necessary, select department, course, start and end dat. After that click on save.
