How an assignment can be deleted?

You can delete an assignment at any time. If an assignment is deleted after the students have submitted their work, all submitted work will also be deleted.

  1. Click the module access icon > Academics > Assignment to open the All assignments page.
  2. Select the subject for which the assignment was given.
  3. Click the title of the assignment.
  4. Click the Edit button and make the necessary changes.
  5. Click the Delete Assignment button.
