How can I create assignment for a batch?

Note:- Even though all employees can access the Assignment plugin, only employees associated to subjects can create assignments.

  1. Click the module access icon > Academics > Assignment to open the All assignments page.
  2. In the top right, click the Create new assignment button.
  3. In the Title field, enter the name of the assignment.
  4. In the Subject field, select the subject. The students who take the subject are displayed on the right.
  5. Select the students who will need to complete the assignment.
  6. In the Content field, enter the description about the assignment. The description can include detailed instructions for students regarding the assignment, the form in which it should be submitted, the grading criteria, and so on.
  7. Click Choose File to attach a file to the submission.
  8. Select the due date for the assignment.
  9. Click the Create new assignment button.
