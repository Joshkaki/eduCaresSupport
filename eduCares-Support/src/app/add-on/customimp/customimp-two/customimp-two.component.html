Custom import mandatory fields

For a successful custom import, the data in your custom import file must be formatted correctly and must match data in EduCares when necessary.

The following tables lists all mandatory fields for each data type.

Employee admission details:

Mandatory Fields Data Format Description Example and Instructions
*Employee Category Name An active employee category. System admin etc.
*Employee number Alphanumeric, underscore, hyphen and back slash (/). AYS/2018/S001
*Joining date mm/dd/yyyy or dd-mm-yyyy 03/13/2016 or 13-03-2016
*First name Alphanumeric
*Gender m or f
*Employee Position Name An active employee position. System admin etc.
*Employee Department Name An active employee department. System admin etc.
*Date of birth mm/dd/yyyy or dd-mm-yyyy mm/dd/yyyy or dd-mm-yyyy
*Nationality Name Country Name. eg. For an employee from India, enter nationality as India.
Mobile phone Numeric. 9876543210
Email Valid email address. you@example.com

Student admission details:

Mandatory Fields Data Format Description Example and Instructions
*Admission no Alphanumeric, underscore, hyphen and back slash (/). AYS/2018/S001.
*Admission date mm/dd/yyyy or dd-mm-yyyy 03/13/2016 or 13-03-2016
*First name Alphanumeric
*Gender m or f
*Batch Name Course code - Batch name For a course ‘Grade 1’ having course code ‘G1' and a batch named '2018 A’ batch name is G1 - 2018 A
*Nationality Name Country Name. eg. For an employee from India, enter nationality as India.
Email Valid email address. This field is mandatory only if moodle is enabled.

Student guardian details:

Mandatory Fields Data Format Description Example and Instructions
*First name Alphanumeric. This column cannot be modified during bulk edit.
Is Father Y or Leave Blank. If guardian is father, enter Y, leave blank otherwise. This column cannot be modified during bulk edit.
Email Valid email address. This field is mandatory only if moodle is enabled.
Set immediate contact Alphanumeric, underscore, hyphen and back slash (/). Enter admission number of the student to be made immediate contact. eg:-AYS/2018/S001

Student attendance details:

Mandatory Fields Data Format Description Example and Instructions
*Student Admission Number Alphanumeric, underscore, hyphen and back slash(/). AYS/2018/S001.
*Month date mm/dd/yyyy or dd-mm-yyyy 03/13/2016 or 13-03-2016.
*Batch Name Course code - Batch name For a course ‘Grade 1’ having course code ‘G1' and a batch named '2018 A’ batch name is G1 - 2018 A.
*Attendance Status Code Enter attendance status code.
Attendance Save Y or leave blank Enter ‘Y’ to save, leave blank otherwise. The attendance will be saved.

Student exam scores:

Mandatory Fields Data Format Description Example and Instructions
*Exam group name Alphanumeric. Name of the exam which is created for the batch.
*Batch name Course code - Batch name For a course ‘Grade 1’ having course code ‘G1' and a batch named '2018 A’ batch name is G1 - 2018 A.
*Subject code Alphanumeric subject code of the subject which we want to enter the score.
*Student Admission Number Alphanumeric, underscore, hyphen and back slash (/). AYS/2018/S001.
Marks Enter the marks of the subject as decimal.
Grading Level Name Alphanumeric. Enter grade value for Grade type exams. It will be auto generated for mark and grade exams.

Library book details:

Mandatory Fields Data Format Description
*Title Alphanumeric.
*Author Alphanumeric.
*Book number Alphanumeric.

Store item details:

Mandatory Fields Data Format Description
*Item name Alphanumeric.
*Quantity Alphanumeric.
*Unit price Alphanumeric.
*Batch number Alphanumeric.
*Store Code An active Store Code.

Item supplier details:

Mandatory Fields Data Format Description
*Name Alphanumeric.
Contact no Numeric.
*Supplier Type Code An active Supplier Type Code.

Store details:

Mandatory Fields Data Format Description
*Name Alphanumeric.
*Code Alphanumeric.
*Store Category Code An active Store Category Code.
*Store Type Code An active Store Type Code.
