Is it possible to attendance status as well as placement status for student?

Yes, it is possible to mark attendance and update the placement status. In the registrations page on the day of the placement there is option mark attendance. If the student has attended, then there is the placement tab where you can approve placement to update that the student has been placed. Note: The option is available after the placement date only.
