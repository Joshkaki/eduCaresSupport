How an event can be created in Educares and view it on calendar?

An event can be created using the Event Creation module or using the Calendar by admin or privileged employee. You can create an event that is common to all Educares users, or for a specific batch/department.

Event Creation module:

  1. At the top of your dashboard, click the module access icon > Collaboration > Event Creation to open the Event creation page.
  2. In the Start date and End date field, specify the event start date and end date.
  3. In the Title field, enter the name of the event.
  4. In the Description field, enter a description of the event.
  5. Select the Is holiday? checkbox if the specified date(s) is a holiday. If the event is common to all Educares users:
  6. Select the Event common to all checkbox.
  7. Click the Create event button to create the event.
  8. Click the Confirm button to publish the event.

If the event is not common to all Educares users, do not select the Event common to all checkbox.

  1. Click the Create event button to create the event.
  2. Click Select a batch, select the batches that will receive the event details, and then click the Save button.
    Note: Parents/Guardians of the students set as the emergency contacts will also receive the event details.
  3. Click Select department, select the employee departments that will receive the event details, and then click the Save button.
  4. Click the Confirm button to publish the event.
    Published events are displayed on the Events dashlet of all Educares users.
    Also, if you have configured SMS settings, the recipients will receive an SMS notification.

From the Calendar:

  1. At the top of the dashboard, click the module access icon > Academics > Calendar to open the Calendar.
  2. Hover the mouse over the top left corner of a date, the Add link will appear. Click Add to create an event, and then follow the above procedure.
