How to send an SMS to an employee from Educares portal?

An SMS can be sent to a employee only if the following conditions are met:

  • On the SMS Settings page, the Send SMS to Employees checkbox is selected.
  • The employeeâ€™s profile has the registered mobile number of the employee.
  1. At the top of your dashboard, click the module access icon > Collaboration > SMS > Send SMS > Select the Employees Checkbox to open the Employees page.
  2. Select a department. All Employees from the selected department are listed.
  3. Select the Employee(s) to who you want to send the SMS text messages.
  4. Enter the message. You can send up to 160 characters in a message and this will be counted/charged as one message. If the message contains more than 160 characters, the message is broken and delivered as separate messages, and you will be charged accordingly.
  5. Click the Send SMS button.

Note : If a Employee's profile does not have the mobile number then the Employee name will be greyed out and cannot be selected.
