How can we generate the student day-wise attendance report?

The Day-wise Attendance Report will display the number of students present in your institution, as well as the number of students absent in each batch on a particular day.
The Day-wise Attendance Report is available only if you mark student attendance Daily.

Only the administrator or a privileged employee (the Student Attendance Viewprivilege is required) can view the Day-wise Attendance Report.

Click the module access icon > Academics > Attendance > Day-wise Attendance Report. By default, the report displays the student attendance count of all batches for the current date.


To view the student attendance count for a previous date, all you need to do is change the date. The Overall Strength in the top right is the total number of students present in your institution for the selected date.

Use the Select a course field to view the number of students absent in each batch of a particular course. You can then click on a batch to view the names of absent students.


To generate a CSV report of the number of students absent in each batch, click the CSV report button.
