What are the different types of grade book report that can be generated in a planner?

Once the required exams are added and marks are entered for a term we can generate the report for the exam/term/ academic year.
Select the class for which report needs to be generated.
There are three types of report that can be generated in Educares.

Exam Report - Report for a particular exam.

Term Report - Report for a particular term. Create term exam before you generate the term report.

Planner Report - For the entire academic year. Planner exam should be created before generating planner report.
