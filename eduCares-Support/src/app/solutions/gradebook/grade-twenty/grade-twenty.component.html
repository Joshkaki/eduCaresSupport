Different types of report cards in Educares?

Once exams are scheduled and mark entered, we can generate report cards in Educares.

Make sure that all the custom settings are completed before you generate the report card.

Different types of report cards that can be generated in Educares

  1. Exam Report:

    Educares lets you generate a report for any exam for which marks have been submitted. This can be done by

    Menu >> Academics >> Gradebook >> Manage Gradebook >> Select Class >> Manage Exams >> Select the exam for which you want the report to be generated >> More >> Generate Report
  2. Term Report:


    To generate term report go to-

    Menu >> Academics >> Gradebook >> Manage Gradebook >> Select Class >> Manage Exams >> Select the term for which report has to be generated >> Actions >> Generate Term Report
  3. Planner Report:

    In Educares we can generate a report card for the entire academic year too and is called a planner report. Make sure the planner exam is created before you try to generate a planner report. This can be done by

    Menu >> Academics >> Gradebook >> Exam Planner >> Choose Exam Plan >> Show details >> Planner Exam


    To generate planner report go to-

    Menu >> Academics >> Gradebook >> Manage Gradebook >> Select Class >> Manage Exams >> Generate Planner Report
