What are Grading Profiles?

Grading scale is any standardized measurement based on which the performance of a student can be assessed.

Before creating exam, it is mandatory to create the different grading profiles used in the school based on which the school evaluates the students’ performance. Grading profile is used in exams which either follow grade system or both Marks & Grades.

Grading profile are of two types –

  • Direct Grades – Grades are not based on marks but is applied directly.
  • Marks based grades – Grades are derived based on marks.

Steps to create grading profile -
1. Menu >> Academics >> Gradebook >>Settings >> Grading Profiles >> Click on create a grading profile

You can either enable the description to be displayed in the report card or disable the description.

2. Add grades
