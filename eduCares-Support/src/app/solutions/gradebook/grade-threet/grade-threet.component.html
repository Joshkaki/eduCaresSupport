How to link a class to gradebook planner?

Once the planner is generated and exams are created for the term, the next step is to link the classes which follows the exam pattern to the planner.

This can be done by following path

Menu >> Academics >> Gradebook >> Exam Planner >> Choose the exam plan >> Mange Classes >> Add classes.
