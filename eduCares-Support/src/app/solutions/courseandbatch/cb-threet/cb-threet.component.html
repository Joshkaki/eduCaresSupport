What privileges does a batch tutor have?

If an employee is set as a batch tutor and teaches at least one subject, then that user will get the following below mentioned privileges by default.

  • Attendance marking for the assigned batch.
  • Marks entry privilege in Examination and Gradebook Module.
  • Can manage Student Remarks.

There is no need of giving any extra privileges for that user to mangae the above mentioned tasks. More over the same employee can be assigned as tutor for more than one batches so that he gets the same privilege over the other batch too.

Normally a batch has one group tutor. You may however add more than one tutor in case your institution has a practice of having more than one tutor for a batch.
