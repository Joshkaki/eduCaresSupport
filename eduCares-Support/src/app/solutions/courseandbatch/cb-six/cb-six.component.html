How a batch can be created in EduCares?

To create a batch, you must either be an administrator or a privileged employee (Manage Course/Batch privilege must be provided). Such a user can create different batches under the same course.

At the top of your dashboard, click the module access icon > Administration > Settings > Manage Course/Batch > Manage Course to open the Manage Course page.

  1. Select the course for which you want to create a new batch.
  2. In the top right, click the New button.
  3. In the Name field, enter the batch name. The batch name is automatically prefixed with the course code.
  4. In the Start date and End date field, enter the start date and end date for the batch.
  5. Select the Import previous batch subjects checkbox to assign all previous batch subjects (including elective subjects) to this batch.
  6. Select the Import previous batch master fees checkbox to assign the previous batch fees structures (fee categories and fee particulars) to this batch.
  7. Click the Save button
