What is Batch Transfer? What is the advantage of Batch Transfer?

Batch Transfer allows you to move a set of students from one batch to a new one at the end of an Academic Year. Following this process is necessary to ensure that all the data in this batch is properly saved and made available in the next batch also.

Prerequisite: Create the new batch to which students need to be transferred.

Steps:
1. The link to batch transfer is found in
Settings > Manage Course/Batch > Manage Batch. Click on Batch Transfer


2. Select the Transfer to Batch where you want to do the transfer to. Uncheck any student who is not eligible and hit on Save to complete the process.
