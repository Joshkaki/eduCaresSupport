How to Create / Edit / Delete student fee discount?

Create Fee Discount:

Your institution may offer fee discounts to students. For example, a fee discount may be given to the student whose parent(s) is employed at the institution. Fee discounts can be set up and applied at the time of processing student fee payments. A student fee discount can be created for a batch of students, for a student category, or for specific students.

  1. At the top of your dashboard, click the module access icon > Administration > Finance > Fees > Create Fees > Create Discount.
  2. Select the discount type from the following options:
  • Batch – To create a discount for a batch:
    1. Enter the discount name.
    2. Select a fee category.
    3. Specify if the discount is for batches of the selected category, or for fee particulars of the selected category.
    4. Select the batch(es) or fee particular(s).
    5. Enter the discount amount or percentage based on the discount mode.
    6. Click the Create button.

  • Student Category – To create a discount for a student category:
    1. Enter the discount name.
    2. Select the fee category.
    3. Select the student category.
    4. Specify if the discount is for batches of the selected category, or for fee particulars of the selected category.
    5. Select the batch(es) or fee particular(s).
    6. Enter the discount amount or percentage based on the discount mode.
    7. Click theCreate button.

  • Student – To create a discount for a student:
    1. Enter the discount name
    2. Select a fee category.
    3. Specify if the discount is for batches of the selected category, or for fee particulars of the selected category.
    4. Select the batch(es) or fee particular(s), and then select the students it applies to.
    5. Enter the discount amount or percentage based on the discount mode.
    6. Click the Create button

Edit / Delete Fee discount:
Student fee discounts created for a batch or for a fee particular can be edited only if you haven’t started processing student fee payments. However, you fee discounts can be deleted at any time.

  1. At the top of your dashboard, click the module access icon > Administration > Finance > Fees > Create Fees > Create Discount.
  2. In the top right, click the View Discount button.
  3. Select a batch and a fee category. The list of fee discounts created will be displayed.
  4. Click Edit to edit a fee discount or click Delete to delete a fee discount.
