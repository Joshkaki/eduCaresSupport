Students can be sorted by Admission no., First Name, or Last Name. Based on the sort order, roll numbers can then be assigned for all students of each batch. The student sort order can be changed at any time. If you change the student sort order after assigning student roll numbers, all students will be arranged based on the new sort order, however, the assigned roll numbers will remain the same.

  1. At the top of your dashboard, click the module access icon > Administration > Settings > Manage student roll numbers to open the Manage roll number page.

    2. This page is displayed only the first time when setting the student sort order.
  2. Select the studentsâ€™ sort order.
  3. Click the Save button.
