How to generate report on class/batch details within Educares?

The Class/Batch Details report lists all active courses together with the number of batches and the number of students in each course.

  1. At the top of your dashboard, click the module access icon > Data and Reports > Reports > Class / Batch Details.
  2. Click on to view batches and batch details
  3. Click on a batch to view the students in the batch and student details.

At any time you can click the Export as CSV button to export the details as a CSV file. This step may take a while. When the file status changes from in Queue to Download, click Download and save the file to your system.

Note: Make sure to keep refreshing the web page.
