How can I download the reports? Why it shows the message as 'In Queue' when clicked on Export CSV against a report ?

The reports can be downloaded if you click on Export CSV option, the report generation starts in the background (so that it does not affect your normal tasks). Refresh this page after some time to see the Download link. Click on the Download link to download the csv report. If the message 'In Queue' please try refreshing/reloading the page for some more time.

Since reports in Educares are created by bulk in real time it is showing the message as "In Queue". Typically reports take about a minute or two to get generated. Perform refreshing the page till Download link appears as shown below.
