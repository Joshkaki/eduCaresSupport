How can I search and view a user profile?

The administrator or privileged employees (Manage Users privilege is required) can search for a user from the Manage Users page.

At the top of your dashboard, click the module access icon > Administration > User to open the Manage Users page.

  1. In the Search user field, for an employee, enter the first name, last name, or employee number to search. For a student, enter the first name, last name, or admission number to search.
