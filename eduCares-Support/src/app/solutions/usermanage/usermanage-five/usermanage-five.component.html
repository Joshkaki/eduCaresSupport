What is the use of 'Search by User Type' option in Manage User?

It helps the admin or privileged employees (Manage Users privilege is required) to search for a user based on their roles. At the top of your dashboard, click the module access icon > Administration > User to open the Manage Users page.
In the top right, click the Search by User Type button to open the All Users page.

Select the user role you are searching for from the following options:

  • Admin – Displays all administrators.
  • Employee – Displays all employees for the selected department.
  • Student – Displays all students for the selected batch.
  • Parent – Displays all parents of the students for the selected batch.

Click on the user’s name whose profile you want to view.
