How to use the EduCares Online Meeting Patch?

To get the EduCares patch, you need to follow the below mentioned steps.

  • Log in to the server as root user.
  • Switch to eduCares user
  • Run the following commands
  1. wget https://eduCares:foradian@downloads.eduCares.com/eduCares_patch.sh
  2. chmod +x eduCares_patch.sh
  3. ./eduCares_patch.sh

Add Normal subjects to a batch: To add a normal subject to a batch, you must either be the administrator or a privileged employee (Manage Course/Batch or Subject master privilege..
