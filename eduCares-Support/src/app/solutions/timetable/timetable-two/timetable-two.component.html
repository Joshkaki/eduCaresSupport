How to create class timing sets?

A class timing set defines the number of periods/Class and the duration of each period/Class in a working day. You can create as many class timing sets.
If this is the first time you are creating class timing sets when a new batch is created in Educares, a ‘Default’ class timing set is automatically assigned to the batch. The ‘Default’ class timing set must be edited to add class timing records. The ‘Default’ class timing set can also be deleted. If deleted, the subsequently created class timing set will be automatically assigned when a new batch is created.

  1. At the top of your dashboard, click the module access icon > Academics > Timetable > Manage class timing set to open the Manage class timing sets page.
  2. In the top right, click the New button.
  3. In the Name field, enter the name of the class timing set.
  4. Click the Submit button.
  5. Click View class timings beside the newly created class timing set.
  6. Click Add Class Timing.
    • In the Name field, enter the name of the class timing record. For example, Period 1, Period 2.
    • Enter starting and ending time for the class timing record.
    • Select the Is a break checkbox when creating a break, such as lunch break, recess.
    • Click the Save button. You can similarly create as many class timing records as required.

Note: Make sure class timing records do not overlap each other.
