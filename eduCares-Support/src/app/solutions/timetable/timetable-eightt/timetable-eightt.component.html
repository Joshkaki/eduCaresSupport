How can I allocate a classroom?

The administrator or a privileged employee (Manage Allocations or Manage Buildings and Allocations privilege is required) can allocate classrooms for each class based on timetable slots. Follow the below steps to allocate.

  1. At the top of your dashboard, click the module access icon > Academics > Timetable > Class Room Allocation > Allocate to open the Allocate page.
  2. Select the allocation type from the following options: Weekly – Allocate Classrooms for the complete timetable period.

Date specific – Allocate classrooms for a specific date. Use this option when you want to override an existing classroom allocation.

a.If you selected Weekly:

  1. Select the timetable, and then click the View button. Only batches with timetable entries are displayed.
  2. Select the weekday, for eg, Monday.
  3. In the bottom, select the building. Corresponding rooms are displayed.
  4. Select a room, and then drag and drop the room in any one of the classes. You can similarly allocate classrooms for other classes, for each weekday. Allocated classes are highlighted in green.
    If required, you can allocate more than one classroom to a class. To remove an allocated classroom, click ‘x’.
    After allocating classrooms for other classes, the classroom allocation is automatically applied for the complete timetable.

b.If you selected Date specific:

  1. Select the month and year, and then click the View button. Only batches with timetable entries of the selected year and month are displayed.
  2. Select the day of the month for which you want to allot a classroom for a specific class.
  3. In the bottom, select the building. Corresponding rooms are displayed.
  4. Select a room, and then drag and drop the room in the class record. If required, you can allocate more than one classroom to a class.
