How an employee subject association details report is helpful?

The Employee subject association Details report lists all the active employees and their associated subjects in a batch. This report will help you to keep the track of the Subjects an employee is handling.
The administrator or a privileged employee (View Additional Reports privilege is required) can Export the report in CSV format from the Reports module.

At the top of your dashboard, click the module access icon > Data and Reports > Reports > Employee subject association Details.

  1. Click the Export as CSV button to export the details as a CSV file.
  2. When the file status changes from In Queue to Download, click Download to download and save the file to your system. This step may take a while.
    Note: Make sure to keep refreshing the web page.

Note: A CSV file can be opened using a Application such as Microsoft Excel, LibreOffice.
