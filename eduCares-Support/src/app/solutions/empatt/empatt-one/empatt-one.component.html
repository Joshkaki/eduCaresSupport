How to add different leave types in Educares ?

Leave types are the types of leaves that can be taken by an employee, such as sick leave, casual leave etc., You can add multiple leave types applicable to your institution. For each leave type, you can define the leave count that is to be credited, the leave carry forward and additional leave settings.

At the top of your dashboard, click the Module access icon > Administration > Human Resource > HR Settings > Leave Types > Add Leave Types to open the Add Leave Types page.

  1. In the Leave name field, enter the name of the leave type.
  2. In the Leave code field, enter the leave code.
  3. In the Leave count field, enter the maximum number of leaves an employee can take for this leave type.
  4. In the Valid From field, select the date from which a leave type will be added for all employees.
  5. Set how to manage Employee leave balance during leave reset:
  6. Allow leave carry forward – Enabling this option will be carrying forward the balance leave count for the employees while resetting leave counts. Having two options - either carry forward all balance leaves or only specific count.
  7. Discard leave balance – Will not be carrying forward the balance leave count for the employees. Will be discarding all balance count.
  8. Set how to manage Additional leaves.
  9. Liable for salary deduction (LOP) – Additional leaves can be marked as loss of pay and would be liable to be accounted (added to a payslip) when processing payroll.
  10. No salary deduction – Additional leaves will not be marked as loss of pay.
  11. Select the status of the leave type:
  12. Active – When applying for leave/marking employee attendance, the leave type will be available to select.
  13. Inactive – The leave type will not be available in the option at the time of applying/marking of leaves.
  14. Click the Create button.
